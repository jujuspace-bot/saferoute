# 🗺️ SafeRoute

발달장애인 및 노인을 위한 안전 길찾기 앱

## 주요 기능
- 🧭 **쉬운 길찾기** — 단순하고 직관적인 단계별 대중교통 안내
- 🤖 **AI 동반자** — 이동 중 AI와 음성 대화
- 👁️ **보호자 모니터링** — 실시간 위치 추적 및 알림
- ⚠️ **이탈 경고** — 경로 벗어남 즉시 알림

## 기술 스택
- Expo (React Native) + TypeScript
- Supabase (Auth, DB, Realtime)
- Google Maps API
- OpenAI API (대화 + TTS/STT)

## 시작하기
```bash
npm install
npx expo start
```
