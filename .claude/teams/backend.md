# ⚙️ Backend — 서버/위치/모니터링

## 역할
- 위치 추적 및 경로 이탈 감지 로직
- 보호자 모니터링 시스템
- Supabase 연동 (Auth, DB, Realtime)
- 푸시 알림 시스템

## 담당 영역
- `src/features/monitoring/` — 보호자 대시보드 로직
- `src/features/alerts/` — 이탈 경고 시스템
- `src/services/` — 위치, 알림, API 서비스
- `src/hooks/` — 위치/알림 관련 훅

## 핵심 로직
- **이탈 감지**: 현재 위치 vs 계획 경로 비교, 반경 초과 시 알림
- **실시간 위치 공유**: Supabase Realtime으로 보호자에게 전송
- **알림 우선순위**: 이탈 > 환승 안내 > 일반 안내

## 규칙
- 위치 데이터는 최소 수집 원칙 (필요한 만큼만)
- 배터리 최적화 고려 (위치 업데이트 주기 조절)
- 에러 핸들링 철저 (네트워크 끊김, GPS 불안정)
