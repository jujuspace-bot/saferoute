# 🤖 AI — 대화/음성 시스템

## 역할
- AI 대화 기능 구현 (이동 중 동반자)
- 음성 입출력 (STT/TTS)
- 경로 안내 음성 생성
- 상황 인식 대화 (현재 위치 기반)

## 담당 영역
- `src/features/ai-chat/` — AI 대화 UI + 로직
- `src/services/ai.ts` — OpenAI API 연동
- `src/services/voice.ts` — TTS/STT 서비스

## 대화 설계
- **톤**: 친근하고 안심시키는 말투
- **길이**: 짧고 명확한 문장
- **컨텍스트**: 현재 위치, 남은 거리, 다음 행동 포함
- 예시: "잘 가고 있어요! 다음 정류장에서 내리면 돼요. 2분 정도 남았어요 😊"

## 음성 가이드
- TTS 속도: 일반보다 느리게 (0.8x)
- 중요 안내는 반복 (환승, 하차)
- 백그라운드에서도 음성 안내 지속

## 규칙
- AI 응답은 항상 3문장 이내
- 어려운 단어 사용 금지
- 위치 정보는 Backend 서비스에서 받아 사용
